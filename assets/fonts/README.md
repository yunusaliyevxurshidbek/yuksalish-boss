# Fonts Directory

Place the Plus Jakarta Sans font files here:

- PlusJakartaSans-Light.ttf (300)
- PlusJakartaSans-Regular.ttf (400)
- PlusJakartaSans-Medium.ttf (500)
- PlusJakartaSans-SemiBold.ttf (600)
- PlusJakartaSans-Bold.ttf (700)
- PlusJakartaSans-ExtraBold.ttf (800)

Download from: https://fonts.google.com/specimen/Plus+Jakarta+Sans

Alternatively, the app uses google_fonts package which will download fonts automatically.
